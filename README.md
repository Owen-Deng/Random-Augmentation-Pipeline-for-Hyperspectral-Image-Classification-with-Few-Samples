
Project Code for the Paper:
“Random Augmentation Pipeline for Hyperspectral Image Classification with Few Samples”

This repository contains the official implementation of the methods presented in the journal article listed below. If you use this code in your research or find it useful, please kindly cite the paper.

### Citation

Wang, Y., Deng, C., & Zuo, F. (2022). Random augmentation pipeline for hyperspectral image classification with few samples. International Journal of Remote Sensing, 43(18), 6581–6604. https://doi.org/10.1080/01431161.2022.2132123
